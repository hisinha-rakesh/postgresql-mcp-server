===============================================================================
  TROUBLESHOOTING
===============================================================================

Problem: MCP server not showing in Claude Desktop
Solution:
  1. Check config file path: %APPDATA%\Claude\claude_desktop_config.json
  2. Verify JSON syntax is valid
  3. Completely restart Claude Desktop
  4. Check Claude Desktop logs

Problem: Database connection errors
Solution:
  1. Run test: .venv\Scripts\python test_mcp_setup.py
  2. Verify DATABASE_URL is correct
  3. Check Azure firewall allows your IP

Problem: Python not found
Solution:
  1. Verify path: C:\Users\kusha\postgresql-mcp\.venv\Scripts\python.exe
  2. Make sure file exists
